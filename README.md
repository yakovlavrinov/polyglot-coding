# polyglot-coding

Этот репозиторий — мой личный **playground** для изучения новых языков программирования, технологий и подходов.

Здесь я пробую новое, расширяю кругозор, привыкаю к синтаксису и особенностям разных стеков через небольшие, но осмысленные мини-проекты.

Цель — не собрать огромные приложения, а иметь чистые, понятные и масштабируемые блоки кода, к которым можно быстро вернуться, сравнить реализации или использовать как заготовку.

## Структура

Репозиторий организован по языкам/технологиям. В каждой папке — независимые мини-проекты с понятными названиями.

### Node.js (`node/`)

Практика работы с базовыми возможностями Node.js:

- **node-files-practice**
  - Работа с файлами и потоками:
    - Чтение и запись файлов (`readFile`)
    - Потоковое чтение больших файлов (`createReadStream`)
    - Копирование файлов через пайпы (`copyUsePipe`)
  - Запуск примеров: `node index.js`, `npm start` или отдельные скрипты

### Rust (`rust-lang/`)

Изучение основ Rust через простые программы:

- Базовый "Hello, World!" и эксперименты
- Стандартная структура Cargo-проекта
- Легко добавлять новые модули и примеры

## Планы на будущее

Планирую добавить:

- **Go**  
  Простые примеры concurrency: goroutines, channels, worker pools, параллельная обработка задач
- **Python**  
  Полезные скрипты: парсеры (requests + BeautifulSoup), работа с CSV/JSON, простая обработка данных
- **C#**  
  Основы языка, небольшие консольные примеры, возможно базовый геймдев (консольные игры или простые 2D через сторонние библиотеки)
- **C++**  
  Работа с памятью (указатели, smart pointers), базовые алгоритмы, использование STL (vector, map, algorithms)
- **Сравнения**  
  Одна и та же задача (например, чтение файла, сортировка, concurrent map-reduce) реализованная на нескольких языках для прямого сравнения подходов

## Почему это полезно

- Быстрое погружение в новый язык без создания большого проекта
- Чистые примеры, которые можно копировать и модифицировать
- Личная шпаргалка по синтаксису и идиомам
- Портфолио разнообразия навыков
