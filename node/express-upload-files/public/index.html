Тестирование загрузки файлов в Express + Multer

1. На диск (DiskStorage)

2. В память (MemoryStorage — опасно для больших файлов!)

3. Стриминг (в текущей реализации — тоже в память)

4. Безопасная загрузка (с лимитами и фильтром)

Просмотр загруженных файлов:

  • /uploads — DiskStorage
  • /files — MemoryStorage
  • /stream-files — Streaming
